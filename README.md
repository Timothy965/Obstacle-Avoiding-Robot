# Obstacle-Avoiding-Robot
Learn how to make your own Basic Obstacle Avoiding Robot
Time Needed: 3-4 Hours(if you are solo)
Parts/Components Needed: Ultrasonic Sensor-HC SR04, Arduino Uno, Motor Driver Module- L293D, Breadboard, Chassis, 2x Geared Motors with wheels, Jumper Wires: Male-Male, Male-Female, And x2 9V Batteries / 9-12V DC power supply.
Arduino IDE Software necesssary for uploading the code to arduino
